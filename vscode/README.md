# Visual Studio Code

## Shortcuts

* To search in the global variables in the file. `Ctrl+Shift+O`

* Navigate back and forward(Ex: from definitons).
  * `Alt + Left Arrow`
  * `Alt + Right Arrow`

* Move line Up
  * `Alt + Up Arrow`

* Move line Down
  * `Alt + Down Arrow`

* Remove current line or selected lines
  * `Ctrl + Shift + K`
